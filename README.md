# 📊 Data Analytics Dashboard using Databricks

## 🚀 Project Overview

This project focuses on transforming raw data into meaningful insights using Databricks and SQL.
An interactive dashboard is developed to track key performance indicators (KPIs), analyze trends, and support data-driven decision-making.

---

## 🛠️ Tech Stack

* **Databricks SQL** – Data querying and transformation
* **Python** – Data cleaning and exploratory data analysis
* **Dashboard Visualization** – KPI tracking and insights
* **GitHub** – Version control and project showcase

---

## 📂 Project Structure

Databricks-Project/
